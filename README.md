# NoPVP
- Simple plugin to disable the pvp in worlds

# Commands
Command | Description | Aliases
--- | --- | ---
`/nopvp` | `NoPVP Command` | `/np`

# Features
- You can change message from ```data.yml``` and you can disabled it just set to ""
